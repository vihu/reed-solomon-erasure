reed_solomon_erasure
=====

Elixir rust bindings for reed-solomon-erasure coding

Install
-----

```elixir
def deps do
  [
    {:reed_solomon_erasure, "~> 0.1.0"}
  ]
end
```

Build
-----

$ mix deps.get; mix compile
